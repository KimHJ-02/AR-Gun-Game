# AR Zombie Shooting Game with Hand-Gun Gesture Recognition

## 1. 프로젝트 개요

본 프로젝트는 컴퓨터 비전 기술을 활용하여 사용자의 손 모양을 인식하고, 실시간 카메라 영상 위에 AR 형태의 좀비를 띄워 상호작용하는 게임을 구현하는 것을 목표로 한다.

사용자는 실제 장난감 총이나 컨트롤러 없이 손으로 총 모양을 만들고, 카메라 앞에서 조준 동작을 수행한다. 시스템은 딥러닝 모델을 이용해 손 총 모양을 인식하고, 화면에 등장하는 AR 좀비를 조준하여 공격할 수 있도록 한다.

즉, 본 프로젝트는 다음 기술들을 결합한 컴퓨터 비전 기반 인터랙티브 게임이다.

- 실시간 카메라 입력 처리
- 손 모양 이미지 데이터 수집
- 손 총 모양 분류를 위한 딥러닝 모델 학습
- 손 제스처 인식
- AR 객체 표시
- 좀비 타격 판정 및 게임 로직 구현

---

## 2. 프로젝트 목표

본 프로젝트의 주요 목표는 다음과 같다.

1. 사용자의 손 총 모양을 인식하는 딥러닝 모델을 학습한다.
2. 실시간 웹캠 영상을 입력받아 손 제스처를 판단한다.
3. 카메라 화면 위에 AR 형태의 좀비를 표시한다.
4. 사용자가 손 총 모양으로 좀비를 조준하면 공격 판정을 수행한다.
5. 좀비가 피격되면 사라지거나 체력이 감소하는 간단한 게임 시스템을 구현한다.

---

## 3. 프로젝트 아이디어

일반적인 슈팅 게임은 마우스, 키보드, 게임패드와 같은 입력 장치를 사용한다. 하지만 본 프로젝트에서는 사용자의 손 자체를 입력 장치로 사용한다.

사용자가 손가락으로 총 모양을 만들면 카메라는 이를 실시간으로 인식하고, 해당 상태를 “공격 가능 상태”로 판단한다. 이후 손의 위치나 방향을 이용해 화면상의 좀비와 충돌 여부를 계산하고, 좀비가 맞았는지 판정한다.

이를 통해 사용자는 별도의 장비 없이 카메라 앞에서 손 동작만으로 AR 슈팅 게임을 체험할 수 있다.

---

## 4. 사용 기술

본 프로젝트에서 사용할 예정인 기술은 다음과 같다.

### 4.1 OpenCV

OpenCV는 실시간 카메라 영상 입력과 이미지 처리를 위해 사용한다.

주요 사용 기능:

- 웹캠 영상 입력
- 프레임 단위 이미지 처리
- 이미지 크기 조정
- 화면 출력
- AR 객체 이미지 합성
- 타격 위치 표시

### 4.2 딥러닝 기반 손 제스처 분류

손 총 모양을 인식하기 위해 직접 수집한 손 이미지 데이터를 이용하여 딥러닝 모델을 학습한다.

예상 분류 클래스:

- Gun gesture: 손으로 총 모양을 만든 상태
- Non-gun gesture: 일반 손 모양 또는 다른 제스처

또는 필요에 따라 다음과 같이 클래스를 확장할 수 있다.

- Gun
- Open hand
- Fist
- None / Background

### 4.3 CNN 모델

손 이미지 분류를 위해 CNN 기반 모델을 사용할 예정이다.

가능한 모델 구조:

- 직접 설계한 간단한 CNN
- MobileNet 기반 transfer learning
- ResNet 기반 transfer learning

프로젝트 초기 단계에서는 간단한 CNN 모델로 시작하고, 성능이 부족할 경우 pretrained model을 활용한 transfer learning을 적용한다.

### 4.4 AR 객체 표시

카메라 영상 위에 좀비 이미지를 합성하여 AR 게임 화면을 구성한다.

예상 구현 방식:

- 카메라 프레임 위에 좀비 이미지 overlay
- 좀비 위치 랜덤 생성
- 일정 시간마다 좀비 이동
- 좀비 피격 시 사라지거나 점수 증가

## 5. 데이터 수집 계획

손 총 모양 인식을 위해 직접 이미지 데이터를 수집한다.

### 5.1 수집 데이터

다양한 환경에서 손 이미지를 촬영한다.

수집 조건:

- 다양한 손 위치
- 다양한 손 크기
- 다양한 배경
- 다양한 조명
- 좌우 손 모두 포함
- 손 총 모양과 일반 손 모양 모두 포함
